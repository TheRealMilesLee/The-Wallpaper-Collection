### 🔧 **File Format Guide | 文件格式指南**

#### **📄 Format Specifications | 格式说明**
**English:**
- **HEIC**: Apple ecosystem exclusive, requires compatible viewer
- **PNG**: Lossless compression, ideal for simple graphics and transparent backgrounds
- **JPG/JPEG**: Lossy compression, suitable for photography, smaller file sizes
- **Recommendation**: macOS users can use directly, Windows users may need format conversion tools

**中文:**
- **HEIC**: Apple生态专用格式，需要支持的查看器
- **PNG**: 无损压缩，适合简单图形和透明背景
- **JPG/JPEG**: 有损压缩，适合摄影作品，文件较小
- **建议**: macOS用户可直接使用，Windows用户可能需要格式转换工具
